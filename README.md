# 🎬 MovieApp — API de Películas

Aplicación web desarrollada con **React + Vite** que consume una API de películas para mostrar información como títulos, descripciones e imágenes.

🚀 Proyecto enfocado en mejorar habilidades de frontend moderno y consumo de APIs.

---

## 🔥 Demo

👉 Desarrollado por **Jorge Nuñez**

🔗 Repositorio:  
https://github.com/JorgeUrbina20/api-pelicula

---

## 🚀 Tecnologías utilizadas

- ⚛️ React  
- ⚡ Vite  
- 🎨 Tailwind CSS  
- 🌐 Fetch API  
- 🔀 React Router  

---

## 🧩 Características

✅ Búsqueda de películas en tiempo real  
✅ Navegación por categorías (Acción, Comedia, Terror)  
✅ Consumo de API externa (TMDb)  
✅ Modo oscuro (Dark Mode) 🌙  
✅ Carousel personalizado (sin librerías externas)  
✅ Diseño responsive  
✅ Código limpio y organizado  

---

## 📸 Funcionalidades principales

- 🔎 Buscar películas por nombre  
- 🎬 Visualizar tarjetas con información  
- 🌙 Cambiar entre modo claro y oscuro  
- 🔄 Navegación dinámica con React Router

## ⚙️ Instalación y ejecución

```bash
# 1️⃣ Clonar repositorio

# 2️⃣ Entrar al proyecto
cd api-pelicula

# 3️⃣ Instalar dependencias
npm install

# 4️⃣ Ejecutar proyecto
